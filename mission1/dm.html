Fiche métier : développeur mobile


Missions

Comme son nom l'indique, le développeur mobile se concentre sur le développement d'applications pour smartphones (et tablettes). Un développeur mobile peut se spécialiser sur un OS en particulier ou être hybride.

Compétences

Le développeur mobile doit non seulement connaître l'OS sur lequel il travaille, les bases du développement web, mais il doit également maîtriser divers langages liés à cet OS (Objectif-C est spécifiquement créé pour les appareils Apple, par exemple).
